# 🌐 Bhuvaneswari T — Professional Software Developer Portfolio

Welcome to the source code of my professional software developer portfolio. This is a clean, modern, single-page website built to showcase my projects, technical skills, certifications, and educational background.

## 🚀 Tech Stack & Design

*   **Structure:** Semantic HTML5
*   **Styling:** Custom Vanilla CSS3 (featuring glassmorphism layouts, accent gradients, and smooth scroll transitions)
*   **Interactivity:** Vanilla JavaScript (ES6+)
*   **Icons:** Lucide Icons (injected dynamically via CDN)
*   **Typography:** Google Fonts ("Outfit" and "Plus Jakarta Sans")

---

## 🌟 Key Features

1.  **Glassmorphic Dark Theme:** A premium visual layout using soft transparency, borders, and floating gradient backdrop blobs.
2.  **Dynamic Typewriter Header:** Automatically cycles through profile tags (Java Full-Stack Developer, BCA Graduate, Software Developer, Spring Boot & React Developer).
3.  **Category Project Filters:** Smooth client-side filtering enabling users to view projects by category (All, Java & Spring, React, Vanilla JS).
4.  **Local Resume Integration:** High-reliability view and download options linking directly to `Resume.pdf` in the root folder, loading natively in a new browser tab.
5.  **Interactive Form Simulation:** Captures client contact inputs, logs them to the developer console, and displays a customized validation feedback pop-up.

## 💻 Local Setup & Run Guide

This project is entirely serverless and run-ready with zero build processes or terminal requirements:

1.  **Clone or Copy:** Place the `portfolio` folder anywhere on your computer.
2.  **Add Your Resume:** 
    *   Copy your professional resume in PDF format.
    *   Paste it inside this `portfolio` folder.
    *   Rename the file exactly to **`Resume.pdf`** (case-sensitive).
3.  **Run the Site:** Double-click on **`index.html`** to open it instantly inside any modern web browser (Google Chrome, Microsoft Edge, Safari, Firefox).
